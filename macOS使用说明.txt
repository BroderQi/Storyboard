================================================================================
                    分镜板 Storyboard - macOS 使用说明
================================================================================

欢迎使用分镜板！本软件基于 .NET 8 开发，需要安装 .NET 8 运行时才能运行。

--------------------------------------------------------------------------------
首次使用步骤：
--------------------------------------------------------------------------------

1. 安装 .NET 8 运行时

   方式一：使用安装包（推荐）
   - 双击运行文件夹中的：dotnet-runtime-8.0.x-osx-x64.pkg（Intel 芯片）
     或 dotnet-runtime-8.0.x-osx-arm64.pkg（Apple Silicon M系列芯片）
   - 按照安装向导提示完成安装
   - 可能需要输入管理员密码

   方式二：使用 Homebrew
   - 打开终端（Terminal）
   - 输入命令：brew install dotnet@8
   - 等待安装完成

2. 赋予执行权限（首次运行必需）
   - 打开终端（Terminal）
   - 进入软件所在目录
   - 输入命令：chmod +x Storyboard
   - 或者右键点击 Storyboard，选择"打开"

3. 运行软件
   - 双击 Storyboard 即可启动软件
   - 如果提示"无法打开，因为无法验证开发者"：
     * 按住 Control 键点击应用图标
     * 选择"打开"
     * 在弹出对话框中点击"打开"

--------------------------------------------------------------------------------
后续使用：
--------------------------------------------------------------------------------

安装完 .NET 8 运行时后，以后只需双击 Storyboard 即可直接运行。

--------------------------------------------------------------------------------
常见问题：
--------------------------------------------------------------------------------

Q: 如何确认是否已安装 .NET 8？
A: 打开终端（Terminal），输入：dotnet --list-runtimes
   如果看到 Microsoft.NETCore.App 8.0.x 则说明已安装。

Q: 软件无法启动怎么办？
A: 1. 确认已正确安装 .NET 8 运行时
   2. 确认已赋予执行权限（chmod +x Storyboard）
   3. 检查系统安全设置是否允许运行第三方应用
   4. 尝试在终端中运行：./Storyboard

Q: 如何判断我的 Mac 是 Intel 还是 Apple Silicon？
A: 点击左上角苹果图标 → 关于本机 → 查看"芯片"或"处理器"
   - Intel 芯片：显示 Intel Core i5/i7/i9 等
   - Apple Silicon：显示 Apple M1/M2/M3 等

Q: 需要联网吗？
A: 首次安装 .NET 8 运行时可能需要联网下载组件，
   软件运行时如需使用 AI 功能则需要联网。

--------------------------------------------------------------------------------
系统要求：
--------------------------------------------------------------------------------

- 操作系统：macOS 10.15 (Catalina) 或更高版本
- 处理器：Intel x64 或 Apple Silicon (M1/M2/M3)
- 内存：至少 4GB RAM
- 磁盘空间：至少 500MB 可用空间
